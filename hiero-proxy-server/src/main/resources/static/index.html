Explore the API OpenAPI Spec

Quick start — one command

docker
docker run \
  -e HEDERA_ACCOUNT_ID=0.0.xxxxx \
  -e HEDERA_PRIVATE_KEY=your_private_key \
  -e HEDERA_NETWORK=hedera-testnet \
  -p 8080:8080 \
  ghcr.io/hiero-hackers/hiero-enterprise-proxy:latest

Install the Python SDK

bash
pip install "hiero-enterprise-proxy @ git+https://github.com/hiero-hackers/hiero-enterprise-proxy.git#subdirectory=clients/python"

Use it from Python

python
from hiero_proxy import HieroProxyClient

client = HieroProxyClient(base_url="http://localhost:8080")

# Create an account funded with 10 HBAR
account = client.accounts.create(initial_balance_in_hbar=10)
print(account["accountId"])  # 0.0.12345

# Create a fungible token
token = client.tokens.create("MyToken", "MTK", 1_000_000)
client.tokens.mint(token["tokenId"], 500)

Install the JavaScript SDK

bash
git clone https://github.com/hiero-hackers/hiero-enterprise-proxy.git
cd hiero-enterprise-proxy/clients/javascript
npm install && npm run build && npm link

# Then in your project:
npm link @hiero-hackers/proxy-sdk

Use it from JavaScript / TypeScript

typescript
import { HieroProxyClient } from "@hiero-hackers/proxy-sdk";

const client = new HieroProxyClient({ baseUrl: "http://localhost:8080" });

// Create a topic and publish a message
const topic = await client.topics.create("my event bus");
await client.topics.submitMessage(topic.topicId, "Hello Hiero");

// Deploy a smart contract
const contract = await client.contracts.deploy("0x608060...");
console.log(contract.contractId);

Why use this?

🐳

Docker-first

One command to run. No Java, no Maven, no SDK setup required on your machine.

📦

Official SDKs

Type-safe client libraries for Python and TypeScript. Install from GitHub and start building.

📖

Full OpenAPI spec

Interactive API explorer at /swagger-ui. Auto-generate clients in any language.

🔑

Server-side keys

Key generation and rotation happen on the server. You receive the key once — store it safely.

🛡️

One operator account

Configure once. All transaction fees flow through the operator — your clients stay clean.

⚡

Structured errors

All errors return consistent JSON with status codes — easy to handle in any language.
